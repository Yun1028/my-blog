## Overview

Inspired by the Dragoon unit from StarCraft, this project explores the full workflow
of building a custom reinforcement learning project from scratch.

I used generative AI tools such as GPT and Blender to design the robot, converted it into
a USD asset, and trained it in Isaac Sim. This is my first reinforcement learning project
covering the full pipeline from initial idea and asset creation to environment design,
reward shaping, and locomotion training.

## Stack

- GPT
- Blender
- USD
- Isaac Sim / Isaac Lab
- PPO

## Goal

- Build a custom robot inspired by Dragoon
- Learn the full reinforcement learning workflow
- Train locomotion policies in simulation
- Analyze reward design and exploration behavior
