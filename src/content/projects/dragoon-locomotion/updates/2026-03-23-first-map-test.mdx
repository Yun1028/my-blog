## Overview

In this stage, I tested the robot on the first custom map I created.

To keep the task simple, the environment was designed so that the robot only needed to move forward.
This made it possible to check whether the robot could handle basic terrain changes without requiring a complex policy.

As shown in the video, the robot was able to:

- move straight forward
- climb stairs
- cross a simple hurdle

This result confirmed that the robot could already handle several basic obstacles on a custom map.
